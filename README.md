# ESP32-Based IoT Environmental Monitoring System

## Project Overview

This project is a real-time IoT-based environmental monitoring system developed using ESP32 and multiple sensors. The system collects environmental and safety-related data and transmits it to the Blynk cloud platform for remote monitoring through a mobile dashboard.

## Objectives

- Monitor environmental conditions in real time.
- Enable remote access to sensor data using IoT technology.
- Improve safety through gas leak detection and level monitoring.
- Demonstrate integration of multiple sensors with cloud connectivity.

## Features

- Real-time Temperature Monitoring
- Real-time Humidity Monitoring
- Gas Leakage Detection
- Soil Moisture Monitoring
- Water Level Monitoring
- Wireless Data Transmission
- Mobile Dashboard Visualization using Blynk

## Hardware Components

- ESP32 Development Board
- DHT22 Temperature and Humidity Sensor
- MQ-2 Gas Sensor
- Soil Moisture Sensor
- HC-SR04 Ultrasonic Sensor
- Breadboard
- Jumper Wires
- Power Supply

## Software Tools

- Arduino IDE
- Blynk IoT Platform
- Embedded C/C++
- ESP32 Libraries

## System Workflow

1. Sensors collect environmental data.
2. ESP32 processes the sensor readings.
3. Data is transmitted through Wi-Fi.
4. Blynk cloud receives the data.
5. Users monitor the parameters using a mobile dashboard.
